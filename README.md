# lumen
Ambiente de desenvolvimento PHP com docker, lumen, rabbitMq.

Instalar e configurar um ambiente de desenvolvimento PHP utilizando docker.
instalar o Lumen e configurá-lo para abrir uma conexão junto ao RabbitMq.

Para esse projeto é preciso:
- Docker instalado;
- Docker Compose;
- Git;

Nesse projeto, vou seguir a seguinte estrutura:

app/
- .docker
- - Dockerfile
- - vhost.conf
- index.php
- docker-compose.yml
